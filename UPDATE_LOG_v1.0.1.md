### 2️⃣ 安全篩檢新增「無」選項 (Add "None" Option)

#### 之前 (3 個選項)
```
□ 神經症狀
□ 劇烈疼痛
□ 其他警訊
```

#### 現在 (4 個選項)
```
□ 神經症狀
□ 劇烈疼痛
□ 其他警訊
□ 無 ← 新增！
```

#### 「無」選項說明
- **標籤**: 無
- **描述**: 我沒有上述任何症狀，可以開始運動
- **用途**: 讓使用者明確表示沒有症狀，應該通過篩檢

#### 使用者流程改善
```
之前:
  如果沒有勾選任何項目
  → 使用者可能懷疑是否評估正確

現在:
  如果勾選「無」
  → 明確表示已檢查，沒有症狀
  → 應用程式顯示「✅ 安全通過」
```
